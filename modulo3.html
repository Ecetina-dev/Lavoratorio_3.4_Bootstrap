HTTP (HyperText Transfer Protocol) es el protocolo de comunicación base de la World Wide Web. Define cómo los mensajes son formateados y transmitidos entre un cliente (navegador) y un servidor web.

Métodos HTTP principales:
GET
Solicitar datos del servidor (solo lectura).
POST
Enviar datos al servidor para crear un nuevo recurso.
PUT
Actualizar un recurso existente por completo.
DELETE
Eliminar un recurso del servidor.

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para la creación de páginas web. Proporciona la estructura semántica del contenido, utilizando etiquetas para diferenciar entre encabezados, párrafos, enlaces y contenido multimedia.

Estructura Básica HTML5: 

<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Mi primera página</title>
</head>
<body>
    <h1>Hola Mundo</h1>
    <p>Este es mi primer párrafo.</p>
</body>
</html>

CSS (Cascading Style Sheets) es el lenguaje utilizado para describir la presentación visual de los documentos HTML. Permite separar el contenido de la apariencia, controlando colores, tipografías, márgenes y la adaptación del diseño a múltiples resoluciones de pantalla (Diseño Responsivo).

Bootstrap es un robusto framework de frontend que agiliza el desarrollo web. Proporciona una colección de clases CSS predefinidas, componentes interactivos y un poderoso sistema de grillas (Grid System) que facilita la creación de interfaces de usuario consistentes y adaptables a móviles.

JavaScript es el lenguaje de programación esencial que dota de interactividad y dinamismo a las páginas web. Desde la manipulación del Document Object Model (DOM) hasta la comunicación asíncrona con el servidor, JS es el núcleo del ecosistema frontend moderno.
